# LTX 2.3 — LoRA Training: Voice ID-LoRA & Character Visual Consistency

## Overview

LTX 2.3 supports LoRA fine-tuning for both **audio identity** (consistent voice across clips) and **visual identity** (consistent character appearance). Training uses the official `ltx-trainer` package or ComfyUI custom nodes.

## Voice ID-LoRA

### What it does
Locks in a voice identity using a 5-second reference audio clip. LTX 2.3 generates video with audio where the voice matches the reference for every clip, with native lip-sync. Unlike external audio input (which strips ambient sound), ID-LoRA lets you control dialogue AND background sound simultaneously.

### Pre-trained models (AviadDahan)
- [CelebVHQ-3K](https://huggingface.co/AviadDahan/LTX-2.3-ID-LoRA-CelebVHQ-3K) — 2022 dataset, better generalization, scene variety, background sounds
- [TalkVid-3K](https://huggingface.co/AviadDahan/LTX-2.3-ID-LoRA-TalkVid-3K) — 2025 dataset, more speakers, theoretically supports more speaking styles

### ComfyUI workflow
- By Kijai: [HF Discussion #40](https://huggingface.co/Kijai/LTX2.3_comfy/discussions/40)
- [Comfy.org workflow](https://www.comfy.org/workflows/video_ltx2_3_id_lora-db4717458b6a/)
- Requires up-to-date ComfyUI (ID-LoRA support added recently)
- Inputs: reference image + 5s reference audio + text prompt with dialogue
- ID-LoRA node has a **strength** parameter — increase for higher consistency
- Uses MelBand RoFormer internally to extract clean vocals from reference audio
- Community workflows also available at [RuneXX/LTX-2.3-Workflows](https://huggingface.co/RuneXX/LTX-2.3-Workflows)

### Training your own ID-LoRA
- Dataset: Generate multiple clips of the target voice using Qwen3-TTS (~20-50 clips, varied content)
- Tool: [`ltx-trainer`](https://github.com/Lightricks/ltx-trainer) — official Lightricks tool
- Alternative: Ostris AI Toolkit has a ComfyUI workflow for step-by-step training
- YouTube: [How to Train Custom LTX 2.3 LoRAs (Video + Audio)](https://www.youtube.com/watch?v=Ibw0yoV-Ly0)

### Known limitations
- No native voice speed control parameter
- Workarounds: insert `....` for pauses, provide slower-paced reference audio
- Text-to-speech accuracy improving but not perfect — some words may be unclear

## Visual Character LoRA / IC-LoRA

### What it does
Trains LTX 2.3 to recognize and reproduce a specific character's visual identity across generated video clips.

### LoRA types
| Type | Purpose | Training data |
|------|---------|---------------|
| **Character LoRA** | Consistent character appearance | 10-20 still frames from different angles/poses |
| **IC-LoRA** | Structural control — keeps pose/structure consistent across shots | Video clips or frame sequences showing specific structures/motions |
| **Style LoRA** | Consistent visual style (color palette, art direction) | Collection of styled images |

### Training pipeline
1. **Generate reference images**: Use FLUX.1 Dev (best character consistency) to generate 10-20 images of the character from different angles, poses, expressions
2. **Prepare dataset**: Still frames are simpler than video clips — model learns identity without processing motion
3. **Train**: Use `ltx-trainer` or AI Toolkit ComfyUI workflow
   - GitHub: [Lightricks/ltx-trainer](https://github.com/Lightricks/ltx-trainer) — monorepo with `ltx-core`, `ltx-pipelines`, `ltx-trainer`
   - [WaveSpeedAI training guide (Mar 2026)](https://wavespeed.ai/blog/posts/ltx-2-3-lora-training-guide-2026/)
4. **Deploy**: Load LoRA in ComfyUI alongside the main LTX 2.3 model

### Training resources
- [Ostris: Train LTX-2.3 Character LoRA](https://www.youtube.com/watch?v=JQIl8DFTL1M) — step-by-step tutorial
- [Dual Character Consistency in LTX 2.3 IC LoRA](https://www.reddit.com/r/comfyui/comments/1sl6rue/dual_character_consistency_in_ltx_2_3_new_ic_lora/) — multi-character consistency
- [Easy LoRA training inside ComfyUI](https://www.reddit.com/r/comfyui/comments/1rv9kol/ltx_2_3_easy_lora_training_inside_comfyui/) — custom nodes for step-by-step training

### Recommended approach for character video series
1. Generate reference images with **FLUX.1 Dev** (best character consistency for still images)
2. Train **Character LoRA** on those images for LTX 2.3 visual consistency
3. Train **ID-LoRA** on character voice reference clips (generated by Qwen3-TTS)
4. Use both LoRAs together in ComfyUI for fully consistent character + voice in every video clip
5. This eliminates the need for TTS + ffmpeg audio replacement and achieves native lip-sync with consistent identity

## Hardware Requirements
- LTX 2.3 model: ~47GB total
- Training: benefits from 24GB+ VRAM (RTX 5090 32GB works)
- Must alternate between llama-server and ComfyUI (cannot run simultaneously on same GPU)

## Future Roadmap
- Long-video workflow: sequential videos with overlapping frames to prevent voice identity loss across cuts
- V2V variant: add consistent speech to pre-existing silent videos
- "Dub it" and "Just talk" workflow variants
